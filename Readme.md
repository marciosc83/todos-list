## Tech stack
- Node.js
- Playwright
- Typescript

## How to run the tests
1. Download and install Node.js.
Refer to https://nodejs.org/en/download/current

2. Clone the Project from the Git Repository to your local machine

3. Through the Terminal CLI, enter the Project directory and install the Project dependencies
```
npm install
```

4. Run the tests
```
npx playwright test
```

5. View the tests results
```
npx playwright show-report
```
